IF YOU HAVE UNIVERSAL TRACKER INSTALLED, REMOVE IT FROM YOUR CUSTOM_WORLD FOLDER!!! This APworld uses SOFT PROGRESSION REQUIREMENTS to limit number of checks per sphere, so you will often be out-of-logic!
